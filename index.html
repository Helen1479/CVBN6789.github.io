Rearch Questions

1. What does Quality Assurance mean for web site development?

The Quality Assurance means ensure end users get a functional user interface and the best user experience when using a web application or website.

2. Describe the two key principles of Quality Assurance: "Fit for Purpose" and "Right First Time."

In this Quality Assurance, there are two principles of "Fit for Purpose" and "Right First Time". Firstly, the "Fit for Purpose" indicates the product or service meets its intended purpose. Secondly, the "Right First Time" means avoid errors and any mistakes are immediately dealt with.

3. How do web standard relate to web quality?

Web standards are rules and guidelines developed by the World Wide Web Consortium (W3C) to promote consistency in the design code for composes web pages. The web standard can improve each web quality.

4. Describe what Web Accessibility means. What steps you can take to make your pages more accessible?

The Web Accessibility which Ensures there no barriers to prevent physical disability, contextual disability, and inclusive practices for people whose bandwidth and speed is socioeconomically restricted to interact with or access websites on the World Wide Web. In addition, I can use these three method of steps to make my pages more accessible: 1. Choose a content management system to supports accessibility. 2. Make the link get an unique and descriptive name. 3. Use headings in correctly to organize the structure of each content.
